1 - Primeiro copie a pasta para C:BACKUP DOCUMENTOS 

2 - Execute o arquivo GeraUsersBKP.bat , vai gerar um arquivo com o nome user.lst
	2.1 - Botão direito no user.lst e abra como bloco de notas
	2.2 - Apague os usuários inúteis
	2.3 - Execute o usersBKP.bat para fazer o backup dos usuários que você escolheu

3 - Execute o GeraPastaRaiz para fazer backup da raiz, será gerada um arquivo chamado pasta.txt
	3.1 - Abra ele e coloque deixe somente as pastas que você quer
	3.2 - Execute PastasRaizBKP2.0

OBS:Esta versão não pega app data de winxp, gera o arquivo.7z que pode ser aberto pelo 7z e o backup do email esta melhor.
